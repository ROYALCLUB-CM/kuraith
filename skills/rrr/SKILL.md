# /rrr — Review, Reflect, Record

## Description
ทบทวนวันนี้ → ถอดบทเรียน → บันทึกเป็น diary ลง KURAITH

## Instructions
1. เรียก `kuraith_reflect` เพื่อดูกิจกรรมวันนี้ (รวม pipeline stats)
2. ถาม user ว่าวันนี้ทำอะไร / เรียนรู้อะไร / ติดปัญหาอะไร / พรุ่งนี้ทำอะไร
3. สรุปเป็น diary entry
4. เรียก `kuraith_learn` บันทึก:
   - title: "Daily Review: YYYY-MM-DD"
   - concepts: ["daily-review", "diary"]
   - stage: "refined"
5. ถ้ามี pattern ซ้ำ → ใช้ `kuraith_refine` promote เป็น gem
